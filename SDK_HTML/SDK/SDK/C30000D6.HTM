AH = 05h: Set Display Page

This function sets the active display page. In text modes, pages range from 0 through 7. In all graphics modes the page number must be 0. An "instantaneous" screen change can be made in text modes which support multiple pages by building a screen on an undisplayed page, then using this function to display it.

  • Input: AH = 05h

    • AL = Page number: (0-7)

  • Output: None

  • Error conditions: None

  • Registers modified: AX

Note: Video service functions 06h and 07h affect only display RAM. They are not dependent on the position of the video cursor and have nothing to do with the windowing done in the background to keep the cursor always in view in the zoom text modes.

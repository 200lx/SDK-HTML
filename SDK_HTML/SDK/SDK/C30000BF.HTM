AH = 08h: Set Power-On Alarm

This function sets an alarm that has additional capability over that of an alarm set with function 06h. If an alarm is set with this function, it will power-up the system from deep sleep. If the alarm does not have this capability, the RTC Interrupt Service will treat it the same as one that does, except that the display will not be turned on. Non power-up alarms can provide periodic services (that is, RTC Adjustments) to be invoked in a manner transparent to the user. See "AH = 0Dh: Set Future Power-On Alarm" on page 3-129 for more information.

  • Input: AH = 08h
    CH = Hours in BCD
    CL = Minutes in BCD
    DH = Seconds in BCD

  • Output: AH = 00h

  • Error conditions: Carry Set iff alarm inputs invalid, an alarm is already set, or RTC data is invalid.

  • Registers modified: AX, STATUS

Chapter 1. Introduction

In This Manual

This manual gives you the technical information you may need in working with Hewlett-Packard palmtop products. Part I discusses programming issues, including detailed descriptions of the following:

  • The basic Input/Output system (BIOS).

  • Lotus's System Manager application programming interface (API).\

  • The Lotus handheld application programming interface (LHAPI).

  • The Database engine.

Part II covers HP palmtop hardware and the "Hornet" IC.

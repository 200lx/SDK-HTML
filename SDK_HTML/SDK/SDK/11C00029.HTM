DB_InViewpt

This function tests a record in memory against a given viewpoint definition. 1 is returned if the record meets the specification, otherwise 0 is returned. 

  • Syntax:             DB_InViewpt(DBBLOCK *dblk, char far *record, 
                    VIEWPTDEF far *viewptdef);

  • Returns: 0 Record is not in the viewpoint

    1 Record is in the viewpoint

  • May Move Memory: No

m_clear_api_error

This call clears the last Class/Function error and resets the last Failed Function identifier to -1. In effect, m_clear_api_error implies no failed functions, or cleans up to identify future Failed Function calls.
    

  • Syntax:         int m_clear_api_error()
    • 

  • Parameters:     None
    • 

  • Returns:        The last failed function (dispatch error) identifier
                (similar to the m_get_api_error return)
    • 

  • Related Calls:   m_get_api_error
    •

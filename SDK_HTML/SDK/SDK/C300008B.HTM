AH = 51h: Mailbox Service (Store/Retrieve word at MailBox[i])

This function provides storage and retrieval of any of 8 words in extended BIOS RAM. The built-in applications use the first few of these locations. The others are available. Inputs include a direction flag by specifying Store or Retrieve, and an index specifying the word location in the mailbox. In addition, if the operation is a store, the word to be stored is an input. The service exits CS iff the value in AL is invalid.

  • Input: AH = 51h.
    AL[7] = Direction Flag (0=Recall, 1=Store)
    AL[6;0] = i (Index: 0-7)
    BX = Value to store (if AL[7]=1)

  • Output: CS - Error (Invalid AL)
    CC - If Store, (bx) --> MailBox[i]
    If Recall, (ax) <-- MailBox[i]

  • Registers modified: AX, FLAGS

Key Sequences For Char Codes 80h-8Fh

The following table gives the HP palmtop key sequences to obtain character codes 80h through 0FFh. Those characters not available through an [Fn] sequence may be obtained via the HP palmtop equivalent of the [ALT][NUMPAD] feature present on PC keyboards, which is not yet available on HP palmtops. The character descriptions assume Code Page 850.

In MuteKey sequences that require [SHIFT] to be pressed, the order of the [Fn] and [SHIFT] must be observed. In the others requiring both [Fn] and [SHIFT], the [Fn] and [SHIFT] keys may be pressed in either order, [Fn] first or [SHIFT] first.

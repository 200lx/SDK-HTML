m_get_far_user_handle

This call returns a near handle to a free USER far pointer entry to be used in far pointer addressing. m_get_far_user_handle assumes that block_size is defined in DS segment. 
    

  • Syntax:         m_get_far_user_handle(void) 
    • 

  • Parameters:     None 
    • 

  • Returns:        Returns a pointer (in AX) to an entry in the user's USER portion of the
                registered far pointer table 
                Pointer is actual near offset to far pointer entry.  
                ERROR_TABLE_NOT_REGISTERED - Table not registered 
                ERROR_TABLE_FULL - No room in far table 
    • 

  • Related Calls:  m_get_far_handle

m_free_SVC_entry

This function frees an S_SERVICE entry. 
    

  • Syntax:         m_free_SVC_entry(a) 
    • 

  • Parameters:     a 
    • 

  • Returns:        0 - if entry is successfully freed 
                ERROR_TABLE_ENTRY_RANGE - Pointer invalid 
    • 

  • Related Calls:  See m_get_SVC_entry.  
    •

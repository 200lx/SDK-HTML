m_unload_client

This call unloads the Redirector application. 
    

  • Syntax:        m_unload_client() 
    • 

  • Parameters:    None 
    • 

  • Returns:       0 - if successful 
               Any other value indicates an error condition.  

  • Related Calls: m_load_client 
    • 

  • Comments:      This function is not available on the 200LX.  
    •

m_playback_macro

This function sets a macro to be played back. 
    

  • Syntax:         int far m_playback_macro(int MacroNumber) 
    • 

  • Parameters:     int MacroNumber Number of macro to be played back 
    • 

  • Returns:        Size of retrieved macro 
    • 
                0 - if none assigned 
                ERROR_MACRO_NUMBER (-1) - if illegal macro number 
                ERROR_MACRO_RECORDING (-2) - if macro is currently being 
                                             recorded (macro_save) 
                ERROR_MACRO_PLAYING (-3) - if macro is currently being played 
                                           back (macro_count) 
                ERROR_MACRO_DISABLED (-4) - if macros are disabled (macros_off) 

  • Related Calls:  m_enable_macros.

AH = D3h: Activate Bitmap

This function specifies the bit patterns that will be used by invocations of the BIOS Write BitBlt String function. This service will mark the active Bit Map by setting the high bit in its Status field. It will also clear this bit for other bitmaps currently loaded. See Function D1h for the Bit Map ID Format.

  • Input: AH = D3h
    BX = Bit Map ID

  • Output: If successful, CC
    If unsuccessful, CS

  • Registers modified: AH, FLAGS

  • Error conditions: Specified Bitmap not loaded

m_col_init

This function sets up sort-order tables for applications using the Lotus International Character Set (LICS). This function must be called whenever the sort order is changed. 
    

  • Syntax:         void m_col_init(void)

  • Parameters:     None

  • Returns:        None

  • Comments:       The System Manager supports three different sort orders for 
                applications using LICS: 
                  - Numbers First 
                  - Letters First 
                  - ASCII
                The user may change the sort order by using a menu option in the 
                HP palmtop's Setup application.

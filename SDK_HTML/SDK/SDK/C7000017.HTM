Call Descriptions

Function calls associated with Clipboard Management Services include:

  • m_cb_read Reads data from the clipboard.

  • m_cb_write Writes data to the clipboard.

  • m_close_cb Closes and releases the clipboard.

  • m_fini_rep Signals clipboard that no further data will be sent for this representation.

  • m_get_cb_info Retrieves information on the current state of the clipboard.

  • m_new_rep Prepares the clipboard to receive a new representation.

  • m_open_cb Claims the clipboard, locking out other users.

  • m_rep_index Gets the index and length of a representation.

  • m_reset_cb Clears the contents of the clipboard.

AH = 00h, 01h, 02h, 03h, 04h: Cassette Functions

These functions are mentioned for compatibility reasons. HP palmtops do not support the cassette drive. Register AH is set to 86h and the carry flag is set to indicate an error.

  • Input: AH = 00h, 01h, 02h or 03h

  • Output: AH=86h and CF set to indicate an error

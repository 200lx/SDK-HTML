DB_CardChanged

This function is used to detect a card change event and restore the database handle accordingly, or if the card has changed, shut down the database. DB_CardChanged must be used to properly access databases that are resident on a card, since turning the unit on and off also generates DOS media changed events. See the example event loop on page 11-444 for more information. 

  • Syntax: int DB_CardChanged(DBBLOCK *dblk);

  • Returns: DB_OK No card change or card change detected and corrected or database not open.

    See DB_Reactivate for list of additional errors.

  • May Move Memory: Yes

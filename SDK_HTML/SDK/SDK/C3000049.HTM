AH = 04h: Read Light Pen Position

This function returns the current state and position of a light pen. Since HP palmtops do not support a light pen, this function always returns AH = 0 to indicate the light pen is not activated.

  • Input: AH = 04h

  • Output: AH = Light pen state (0 = not activated)

  • Error conditions: None

  • Registers modified: AX

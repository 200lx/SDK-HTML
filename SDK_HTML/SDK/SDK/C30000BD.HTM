AH = 06h: Set Alarm

This function sets the alarm to generate an Int 4Ah at the specified time. If the alarm time is before the RTC time, the alarm will become active tomorrow, else it becomes active today. The user must place a pointer to an appropriate interrupt handling routine in the Int 4Ah vector. The alarm will be rescheduled to reoccur every 24 hours if not reset using Fcn AH = 07h. (See Also Fcn 0Ch.)

  • Input: AH = 06h
    CH = Hours in BCD
    CL = Minutes in BCD
    DH = Seconds in BCD

  • Output: AH = 00h

  • Error conditions: Carry Set iff alarm inputs invalid, an alarm is already set, or RTC data is invalid

  • Registers modified: AX, STATUS

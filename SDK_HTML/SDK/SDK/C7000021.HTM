Collating Sequence Services

The Collating Sequence services provide character and string comparison functions which may be used to sort items. These comparisons are performed under a Numbers First sorting order, based on the Code Page 850 (CP850) character set.

Separate calls are used for collating services in application development with the Lotus International Character Set (LICS). The LICS calls are described in "LICS Call Descriptions"

AX = 6001h: Measure Backup Battery Voltage

This function measures the voltage of the backup battery. This battery is not used until the main batteries are depleted. Its primary function is to maintain the System RAM disk data. The measurement details and interpretation of the returned value are identical to function 6000h.

  • Input: AX = 6001h

  • Output: AX = Measured value (0-63)
    CARRY set iff one or more samples is "Out-Of-Range"

  • Registers modified: AX,FLAGS

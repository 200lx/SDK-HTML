AH = DBh: Get ROMFONTS

This function copies a built-in ROM-based font structure into a user-supplied buffer. The buffer must be large enough to contain the complete structure.

ROMFONT Size = 5*2 + 6*2*128 + (8+8+11+12)*2*m = 29,392 bytes.

  • Input: AH = DBh
    ES:SI = Pointer to start of supplied buffer
  • Output: Font Structure in supplied buffer

  • Registers modified: AX, FLAGS

General Purpose I/O

The Hornet IC includes 27 "GPIO" pads that are used to interface to external devices completely under software control. On the palmtop computer, these devices include the PCMCIA card port, the display module, and the power supply. These pads can be individually configured as input, output, or as various control output signals. In XCPU mode, most of these pins are dedicated to provide the interface for the external CPU.

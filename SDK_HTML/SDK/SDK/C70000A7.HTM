m_get_SVC_entry

This function gets a pointer to an S_SERVICE entry. 
    

  • Syntax:          m_get_SVC_entry(far *entry_address, far *global_flag) 
    • 

  • Parameters:      far *entry_address - Pointer to location to post SVC entry address 
                 far *global_flag 
    • 

  • Returns:         0 - if entry is assigned successfully 
                 ERROR_SERVICE_TABLE_FULL - Service table full
    • 

  • Related Calls:   m_free_SVC_entry frees an S_SERVICE entry.
                 m_locate_SVC locates a specific S_SERVICE entry.

AX = 6003h: Set Battery Type

This function specifies the type of Main battery installed. HP palmtops support only 2 options here: NICAD, ALKALINE. The type specified will effect charging and end-of-life aspects of system behavior. It also will effect certain displays in the Setup application. This service sets/resets bit 7 (TYPE) of [BATTFL1].

  • Input: AX = 6003h
    BL = Type (0 = Alkaline, Else = Nickel Cadmium)

  • Output: AL: System Power Status (See Fcn 6002h)
    [CX DX]: Charge time (See Fcn 6002h)

  • Registers modified: AX, CX, DX

This function sets or resets bit7 (TYPE) of the RAM Variable BATTFL1.

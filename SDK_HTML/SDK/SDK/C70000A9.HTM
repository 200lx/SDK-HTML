m_load_TSR

This call installs special TSR module into static RAM area. 
    

  • Syntax:      int m_load_TSR(far *filespec_ptr) 
    • 

  • Parameters:  far *filespec_ptr - Pointer to ASCIIZ file specification 
    • 

  • Returns:     0 - if special TSR is successfully installed 
             Any other value indicates an error installing the TSR.  
    •

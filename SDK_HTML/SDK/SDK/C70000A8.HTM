m_load_client

This function loads the Redirector application. 
    

  • Syntax:         m_load_client() 
    • 

  • Parameters:     None 
    • 

  • Returns:        0 - if successful
                Any other value indicates an error condition.  
    • 

  • Related Calls:  m_unload_client 
    • 

  • Comments:       This function is not available on the 200LX.  
    •

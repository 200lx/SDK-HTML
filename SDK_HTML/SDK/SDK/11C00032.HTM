DB_WinTranslate

This function converts the entire file from Windows to Code Page 850. This function is not typically called by the user, but happens automatically when opening an OmniBook file. 

  • Syntax:             DB_WinTranslate(DBBLOCK *dblk);

  • Returns: DB_OK Translate happened okay.

    DB_MEMORY Not enough memory to translate.

  • May Move Memory: Yes

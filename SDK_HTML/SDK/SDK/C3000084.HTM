AX = 4C02h: Return Available Keyboard Types

This function returns the types of keyboards available. The keyboards that are available is set at assembly time (set at the factory as variable KBDS), and may be different for different HP palmtop ROM releases. The format of the return code in AX is the sum of available keyboard codes from among those listed in service 4c00h. For example, a return code of AX=8001h would indicate that US and BELGIUM keyboards are supported.

  • Input: AX = 4C02h

  • Output: AX = available keyboard type(s)

  • Registers modified AX

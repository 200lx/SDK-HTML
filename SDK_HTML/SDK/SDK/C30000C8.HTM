Int 1Dh: Video Parameter Table Pointer

Int 1Dh is a pointer to a standard video parameter table. This table contains parameters that are used to initialize the display controller for a particular display mode in a standard PC. Data for the nonstandard HP 95LX Mode 20h has been removed to conform to the standard table format. An effort is made to make the data in this table meaningful for application programs that may read it. The HP palmtop Display controller requires additional data, and some of the old data in this table is not meaningful. The HP palmtop Display controller is initialized from a separate table that may be accessed by using the Int 10h, Fcn DFh service.

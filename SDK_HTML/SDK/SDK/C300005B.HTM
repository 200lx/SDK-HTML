AH = D4h: Reset Bit Map Stack (Unload All)

This function clears the System Bit Map stack, effectively unloading all of the bit maps available in RAM. The actual Bitmap data is not altered by this function, it simply becomes unreferenced by the Bit Map Stack. However, the next invocation of function D1h will load the specified bit map data at the beginning of the dedicated 16K Bitmap buffer, overwriting any previous data.

  • Input: AH = D4h

  • Output: None

  • Registers modified: AH

  • Error conditions: None

AX = 6004h: Enable/Disable Battery Charging

This function enables or disables the battery charging feature of the HP Adaptor. When the adaptor is plugged in, and charging is enabled, the unit will recharge the installed NICAD batteries. This service sets/resets bit 3 (UCHG) of [BATTFL1].

Warning: battery charging should be enabled only if NiCad batteries are installed. Attempts to charge other types can be hazardous.

  • Input: AX = 6004h
    BL = Enable Charging (0 = Disable, Else = Enable)

  • Output: AL: System Power Status (See Fcn 6002h)
    [CX DX]: Charge time (See Fcn 6002h)

  • Registers modified: AX, CX, DX

This function sets or resets bit3 (UCHG) of the RAM Variable BATTFL1.

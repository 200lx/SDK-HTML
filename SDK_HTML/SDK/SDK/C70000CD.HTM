m_appcount

This call returns the number of other applications currently loaded. 
    

  • Syntax:      int m_appcount(void) 
    • 

  • Parameters:  None 
    • 

  • Returns:     Number of applications currently loaded under System Manager 
    • 

  • Comments:    The Application Manager (MORE task) is not included in this count
             (if the AppMgr is open, this will return the number of apps open -1 ).  
    •

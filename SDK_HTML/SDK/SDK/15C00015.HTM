Display Controller

The Hornet display controller is designed to be MDA and CGA compatible and to support the HP 95LX ("Jaguar") display modes. It is designed to drive a 640 x 200 pixel LCD panel with an 8-bit data bus. A wide variety of fonts are supported which trade off readability for information density and allow tailoring of the aspect ratio. Fonts are stored in system RAM for code page switching and localization. Several options are available for rendering colors as grey shades which trade off visibility for power c onsumption.

The features supported are:

  • Text (Alpha) modes:

      40, 64, or 80 columns with characters 16, 10, or 8 pixels wide.

      Variable number of rows with variable height characters.

      Black and White or 4-shade grey scale.

      MDA/CGA and HP 95LX ("Jaguar") compatible.

      Soft-fonts for nice looking characters, code-page switching, and localization.

      16 Kbyte frame buffer allows 4 pages of 80x25 screens.

      Logical display window can be larger than the physical window, which supports panning and zooming.

  • Graphics modes:

      640 x 200, Black and white, CGA compatible.

      320 x 200, 4 shade grey scale, CGA compatible.

      240 x 128, Black and white, Jaguar compatible (centered in a 320 x 200 display).

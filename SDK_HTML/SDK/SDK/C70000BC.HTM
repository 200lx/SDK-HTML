m_ram_iv_info

This call returns a pointer to the RAM image vector area and the number of active vectors. 
    

  • Syntax:      m_ram_iv_info(void far *a) 
    • 

  • Parameters:  void far *a - Pointer to RAM image vector area 
    • 

  • Returns:     Pointer to RAM image vector area and the number of active vectors 
    •

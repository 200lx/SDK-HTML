Call Descriptions

Function calls associated with Sound Services include:

  • m_asound Generates one of several various sound patterns.

  • m_beep Provides a general purpose sound for error alerts.

  • m_play Plays musical notes in 7 octaves.

  • m_soundoff Turns off the current sound in progress.

  • m_thud Generally used when a user's keystroke cannot be interpreted in the current state.

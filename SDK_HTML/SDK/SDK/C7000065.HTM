m_card_changes

This function returns the count of card changes for the HP palmtop's card slot. This count is incremented whenever a change is detected. Applications can poll this number to determine if the card has been changed since last access. 
    

  • Syntax:      int far m_card_changes(void) 
    • 

  • Parameters:  None 
    • 

  • Returns:     Number of card changes 
    •

AH = 4Dh: Return Model-Specific Information

Returns model-specific information.

  • Input: AH = 4Dh
    AL = D4h

  • Output: BH = 'H'
    BL = 'P'
    CH = Family type (1)
    CL = Model type (2) (HP 95LX=1)
    DH = Submodel Number (HP100LX=0, Reserved=1, HP200LX=2)
    DL = Code Revision

  • Registers modified: AX, BX, CX, DX

The value returned in DL agrees with the "BIOS ROM Version" byte in the System Description Table (See Int 15h, Fcn C0h on page 3-111).

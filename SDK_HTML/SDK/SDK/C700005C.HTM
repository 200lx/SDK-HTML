Call Descriptions

Function calls associated with Event Management Services include:

  • m_action Central branching point for event calls.

  • m_flush_kb Flushes the keyboard queue.

  • m_get_run_mode Returns the current task's run status.

  • m_no_fini Releases the system locks and breaks the shutdown sequence.

  • m_sh_status Gets the current status of the keyboard shift flags.

  • m_yield Voluntary suspension of an application.

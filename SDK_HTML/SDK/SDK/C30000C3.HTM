AH = 0Ch: Set Future Alarm

This function extends the functionality provided by Fcn 06h to allow an alarm to be set at any date in the future. This function, like function 06h will generate an Int 4Ah at the specified time. The user must place a pointer to an appropriate interrupt handling routine in the Int 4Ah vector. The alarm will reoccur every 24 hours if not reset using Fcn AH = 07h.

  • Input: AH = 0Ch
    AL = Year in BCD
    BH = Month in BCD
    BL = Day in BCD
    CH = Hours in BCD
    CL = Minutes in BCD
    DH = Seconds in BCD
    DL = Century in BCD

  • Output: AH = 00h

  • Error conditions: Carry Set iff alarm already set or RTC defunct

  • Registers modified: AX, STATUS

AH = C0h: Return Pointer to System Description Table

This function returns a pointer to the System Description table (SDT) for the HP palmtop.

  • Input: AH = C0h

  • Output: AH = 00h
    ES:BX = Pointer to the System Description Table

  • Error conditions: None

  • Registers modified: AH, BX, ES, FLAGS

The HP palmtop returns 38h as its parameter byte. This indicates that it does not use DMA channel 3 and does not have a cascaded interrupt controller. It does have a real-time clock, supports the Keyboard-Translation-Hook function, and the Wait-For-Event Function.

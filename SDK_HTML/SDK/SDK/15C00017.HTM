Register Descriptions

The display controller registers are mapped into the I/O space at addresses 03B0h-03BFh or 03D0h-03DFh, and at 0022h-0023h. The first two ranges correspond to standard PC-compatible MDA and CGA registers, respectively.

Only one of these ranges may be selected at a time. The last range is where all non-standard hornet registers are accessible via indexing. A portion of these registers are used for display setup and compatibility selection.

Hotkey Sequences

Certain key sequences are processed directly by the code that emulates a hardware keyboard controller (outside of Int 09h). These sequences are provided for display control:

  • [ON][+] Increase display contrast.

  • [ON][-] Decrease display contrast.

  • [ON][*] Shading Technique.

  • [ON][/] Invert Color.

  • [ON][=] Annunciator Control (3 states: Left, Right, Disabled).

  • [MENU] [Arrows] Window movement.

  • [ZOOM] Cycle Zoom State (Text video modes only).

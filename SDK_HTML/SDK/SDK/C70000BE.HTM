m_stop_record

This call terminates the current macro recording. 
    

  • Syntax:         int m_stop_record(void) 
    • 

  • Parameters:     None 
    • 

  • Returns:        The number of the macro currently being recorded 
                0 - if no macro is currently being recorded 
    • 

  • Related Calls:  m_start_record 
    •

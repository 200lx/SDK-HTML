m_keybez

This call checks for built-in international keyboard support. KEYBEZ is a TSR included on the ROM disk that allows the user to configure the 100LX/200LX for international character sets and/or keyboards. 
    

  • Syntax:      int far m_keybez(void) 

  • Parameters:  None 

  • Returns:     1 - if keybez is loaded-- Zero flag reset 
             0 - if keybez is not loaded-- Zero flag set

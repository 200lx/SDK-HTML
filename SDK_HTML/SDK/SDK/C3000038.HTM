Default Color Shading

HP palmtops support several ways to render color information into shades of grey for low resolution graphics modes. The default shading techniques programmed by the BIOS are

  • Mode 4- Frame Rate Shading with Shading Mode 0

  • Mode 5- Bit Dithered Shading with Shading Mode 3

The hotkey sequence [ON][*] toggles between these two shading techniques.

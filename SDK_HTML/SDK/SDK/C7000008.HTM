Call Descriptions

Function calls associated with API Management Services include:

  • m_clear_api_error Sets the last failed function identifier to -1.

  • m_disable_fnct_tbl Disables the FIRST non-disabled function table having the function.

  • m_enable_fnct_tbl Enables the LAST non-disabled function table having the function.

  • m_get_api_error Returns the last failed function identifier.

  • m_get_class_tbl Provides access to API2 dispatch tables.

  • m_get_free_api_class Returns the first available class number in the specified ClassGroup.

  • m_insert_fnct_tbl Inserts a function table into the dispatching mechanism.

  • m_remove_fnct_tbl Removes a function table into the dispatching mechanism.

  • m_search_api_class Searches for an API class.

  • m_valid_api_fnct Returns the status of a function.

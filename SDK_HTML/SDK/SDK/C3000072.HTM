AH = 00h: Initialize Serial Port

This service sets the baud rate, parity, number of stop bits, and character frame size for the specified serial port from data passed in Register AL. The function returns the LSR and MSR registers as shown in the figure below:

    DX = Port# (0 = Built-In Serial Port)

Call Descriptions

Function calls associated with Miscellaneous Services include:

  • m_disable_macros Prevents the editing or display of macros.

  • m_enable_macros Re-enables macro saving/recording after a disable action.

  • m_form_ft Returns a date/time string suitable for display in a directory listing.

  • m_get_sysver Returns the current system version ID.

  • m_playback_macro Sets a macro to be played back..

  • m_ram_iv_info Returns pointer to RAM image vector area and the number of active vectors.

  • m_start_record Stops all macro activity and unconditionally starts macro recording.

  • m_stop_record Terminates the current macro recording.

  • m_sys_rsrc_addr Returns pointer to a string containing system resource address.

  • m_update_apname_list Causes a refresh of the APNAME.LST file for all drives.

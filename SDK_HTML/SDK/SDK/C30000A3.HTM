AH = 03h: Set Typematic Rate and Key Delay

This function sets the typematic rate and delay between key press and when key repeat takes effect. The power-up default value is BX=0103h.

  • Input: AH = 03h
    AL = 05h Set typematic rate
    BH = 00h - 03h for delays of 250ms, 500ms, 750ms, or 1s
    BL = 00h - 1Fh for typematic rates of 25cps down to 1.7cps 
      BL=00h -> 25cps    BL=10h -> 6.4cps 
      BL=01h -> 25cps    BL=11h -> 6.4cps 
      BL=02h -> 25cps    BL=12h -> 5.1cps 
      BL=03h -> 25cps    BL=13h -> 5.1cps 
      BL=04h -> 25cps    BL=14h -> 4.3cps 
      BL=05h -> 25cps    BL=15h -> 4.3cps 
      BL=06h -> 25cps    BL=16h -> 3.7cps 
      BL=07h -> 25cps    BL=17h -> 3.7cps 
      BL=08h -> 13cps    BL=18h -> 3.2cps 
      BL=09h -> 13cps    BL=19h -> 2.8cps 
      BL=0ah -> 13cps    BL=1ah -> 2.6cps 
      BL=0bh -> 13cps    BL=1bh -> 2.3cps 
      BL=0ch -> 8.5cps   BL=1ch -> 2.1cps 
      BL=0dh -> 8.5cps   BL=1dh -> 2.0cps 
      BL=0eh -> 8.5cps   BL=1eh -> 1.8cps 
      BL=0fh -> 8.5cps   BL=1fh -> 1.7cps
  • Output: None

  • Error conditions: None

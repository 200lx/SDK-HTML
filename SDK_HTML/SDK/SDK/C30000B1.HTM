AH = 01h: Initialize Printer

This function initializes a serial printer port. It clears the XOFF received flag, and it does not set the baud rate or parity. This should be done with Int 14h function 00.

  • Input: AH = 01h
    DX = Port number (0 - 1)

  • Output: AH = Printer status (see function AH=00h)

  • Registers modified: AH

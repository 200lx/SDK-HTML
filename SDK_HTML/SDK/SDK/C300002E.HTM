Video Services Interrupts 

AH       STANDARD SERVICES
=====================================================
00h      Set Video Mode 
-----------------------------------------------------
01h      Set Cursor Size
-----------------------------------------------------
02h      Set Cursor Position
-----------------------------------------------------
03h      Read Cursor Position 
-----------------------------------------------------
04h      Read Light Pen Position (Not Supported)
-----------------------------------------------------
05h      Set Display Page 
-----------------------------------------------------
06h      Scroll Window Up 
-----------------------------------------------------
07h      Scroll Window Down 
-----------------------------------------------------
08h      Read Char and Attribute at Cursor Position
-----------------------------------------------------
09h      Write Char and Attribute at Cursor Position
-----------------------------------------------------
0Ah      Write Char at Cursor Position 
-----------------------------------------------------
0Bh      Set Color Palette (Not Supported)
-----------------------------------------------------
0Ch      Write Graphics Pixel
-----------------------------------------------------
0Dh      Read Graphics Pixel 
-----------------------------------------------------
0Eh      Write Teletype Character 
-----------------------------------------------------
0Fh      Get Video State and Mode 
-----------------------------------------------------
13h      Write Teletype String 
-----------------------------------------------------


AH       NON-STANDARD SERVICES 
=====================================================
D0h      Text Zoom 
-----------------------------------------------------
D1h      Load BitMap Data 
-----------------------------------------------------
D2h      Remove (POP) BitMap Data 
-----------------------------------------------------
D3h      Activate BitMap
-----------------------------------------------------
D4h      Reset (UnLoad All) BitMap Stack
-----------------------------------------------------
D5h      Get BitMap Stack Data 
-----------------------------------------------------
D6h      Get BitMap Status 
-----------------------------------------------------
D7h      Bit Block Transfer (BitBlt), Write String 
-----------------------------------------------------
D8h      Load Byte-Wide Font 
-----------------------------------------------------
D9h      Display Context Size/Save/Restore 
-----------------------------------------------------
DAh      Get/Set CodePage 
-----------------------------------------------------
DBh      Get ROMFONTS 
-----------------------------------------------------
DCh      Graphics Cursor Services
-----------------------------------------------------
DFh      Get Video Index
-----------------------------------------------------

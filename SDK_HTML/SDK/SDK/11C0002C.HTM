DB_StoreViewpt

This function saves a new viewpoint definition, and marks the associated viewpoint table as bad. If the viewpoint stored happens to be the current one, a DB_SetCurrentViewpt is executed to keep the viewpoint table in sync. 

  • Syntax:             DB_StoreViewpt(DBBLOCK *dblk, RECORDNUM viewptnum, 
                    VIEWPTDEF far *vpd);

  • Error returns are the same as DB_SetCurrentViewpt.

  • May Move Memory: Yes

m_GetScanCode

This call returns the scancode for a string (mnemonic) representing a key, e.g. Alt-P. 
    

  • Syntax:         unsigned int far m_GetScanCode(char far *szLabelBuffer) 
    • 

  • Parameters:     char far *lszLabelBuffer - Pointer to location to post translated 
                                           scancode 
    • 

  • Returns:        KEY_MAP_OK (0) - for success 
                KEY_MAP_ERROR (-1) - Init resource file results 

  • Related Calls:  m_GetKeyLabel, m_InitKeyLabels 
    •

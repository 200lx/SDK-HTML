DB_CancelViewpt

This function "cancels" the current viewpoint and restores viewpoint 0. Viewpoint 0 is supposed to be the "All items" viewpoint, and is guaranteed to exist for all standard databases. This function is equivalent to a SetCurrentViewpt to 0. 

  • Syntax:             DB_CancelViewpt(DBBLOCK dblk);

  • Error returns are the same for DB_SetCurrentViewpt.

  • May Move Memory: Yes
